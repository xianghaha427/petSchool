# PetSchool 前后端联调指南

## 环境要求

- **Java**: 17+ (当前已安装 Java 24)
- **Maven**: 3.6+ (需要安装)
- **MySQL**: 8.0+
- **Node.js**: 18+ (前端开发)

## 第一步：安装 Maven

### Windows 安装 Maven

1. 下载 Maven: https://maven.apache.org/download.cgi
2. 解压到本地，例如：`D:\apache-maven-3.9.6`
3. 添加环境变量：
   - `MAVEN_HOME = D:\apache-maven-3.9.6`
   - 在 PATH 中添加 `%MAVEN_HOME%\bin`
4. 验证安装：`mvn -version`

或者使用 IDE 内置 Maven（IntelliJ IDEA 自带）

## 第二步：初始化数据库

```bash
# 登录 MySQL
mysql -u root -p11111111

# 执行建表脚本
source D:/PetSchool/packages/backend/src/main/resources/db/schema.sql
```

或者使用 MySQL 客户端工具（Navicat、MySQL Workbench 等）执行 `schema.sql` 文件

## 第三步：启动后端

### 方式一：使用命令行

```bash
cd D:/PetSchool/packages/backend
mvn spring-boot:run
```

### 方式二：使用 IDEA

1. 用 IDEA 打开 `packages/backend` 目录
2. 找到 `PetSchoolApplication.java`
3. 右键 → Run 'PetSchoolApplication'

### 验证后端启动成功

- 访问：http://localhost:8080/api/health
- API 文档：http://localhost:8080/api/doc.html

## 第四步：启动前端

```bash
cd D:/PetSchool/packages/frontend
npm run dev
```

### 验证前端启动成功

- 访问：http://localhost:5173

## 第五步：测试联调

## API 接口列表

| 方法 | 路径 | 说明 | 前端调用 |
| :--- | :--- | :--- | :--- |
| GET | /api/carousel | 获取轮播图 | petService.getCarouselImages() |
| GET | /api/pets | 分页查询宠物 | petService.getPets() |
| GET | /api/pets/{id} | 查询宠物详情 | petService.getPetById(id) |
| POST | /api/pets | 创建宠物 | petService.createPet(data) |
| PUT | /api/pets/{id} | 更新宠物 | petService.updatePet(id, data) |
| DELETE | /api/pets/{id} | 删除宠物 | petService.deletePet(id) |

## 常见问题

### 1. 后端启动失败 - 数据库连接错误

检查 `application.yml` 中的数据库配置：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/pet_school?...
    username: root
    password: 你的密码
```

### 2. 前端调用后端 404

检查 Vite 代理配置 (`vite.config.ts`)：
```typescript
proxy: {
  '/api': {
    target: 'http://localhost:8080',  // 确保端口正确
    changeOrigin: true,
  }
}
```

### 3. 跨域错误

后端已配置 CORS，如果还有问题，检查 `CorsConfig.java`

### 4. 数据类型不匹配

前端类型定义已更新，确保与后端响应格式一致：
- `id`: number (不是 string)
- `gender`: number (1-公，2-母)
- `isVaccinated/isNeutered`: number (0-否，1-是)
